# Memory Bank

This directory stores persistent research knowledge for future agent runs.

Subdirectories:

- `decision-log/`: architecture and method decisions
- `pitfalls/`: repeated mistakes and anti-patterns
- `paper-notes/`: notes from replication papers
- `experiment-lessons/`: postmortems and improvement actions

Guideline:

- Keep entries concise and timestamped.
- Reference relevant experiment IDs and config files.
- Separate objective facts from interpretation.
